## ✨ Features

- 🍩 **Browse Food Categories** — Explore a rich catalog of food items organized by category
- 🧇 **Waffle & Dessert Menus** — Dedicated sections for specialty items
- 🎯 **Action Slider** — Smooth swipe-to-order interaction
- 🎬 **Lottie Animations** — Delightful micro-animations throughout the app
- 🧭 **GoRouter Navigation** — Clean, declarative routing between screens
- ⚡ **Riverpod State Management** — Reactive and scalable app state
- 🎨 **Custom Fonts** — Fredoka One & Poppins for a friendly, modern look
- 📱 **Responsive Design** — Optimized for all screen sizes

## 🚀 Getting Started

### Prerequisites

- Flutter SDK `^3.12.0`
- Dart SDK `^3.12.0`
- Android Studio / VS Code
- An Android or iOS device/emulator

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/dartrox404/ScooPia.git

# 2. Navigate to the project
cd ScooPia

# 3. Install dependencies
flutter pub get

# 4. Run the app
flutter run
```
